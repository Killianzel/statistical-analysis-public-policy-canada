# Statistical Analysis Portfolio – Public Policy 

This repository showcases applied statistical and econometric projects focused on public policy in Canada.

## Projects

### 🎓 Student Retention Analysis
Analysis of retention patterns across post-secondary programs using regression modeling.

### 💉 Immunization Coverage Dashboard
Government of Yukon Public health analysis of MMR vaccination coverage and dashboard design.

### 📊 Education Outcomes Analysis
Econometric analysis of student performance across language instruction programs in Manitoba.

## Tools
- R (dplyr, ggplot2, glm)
- Statistical modeling
- Data visualization
